# budgettracker
A personal budget tracking Android application built with Kotlin as part of the Open Source Coding module.

## Author
Sinoxolo Mkosi

## About the App
Budget Tracker helps users manage their personal finances by tracking expenses across custom categories. Users can register and log in, add expenses with photos, set monthly budget goals, and view spending reports by category.

## Features
- User registration and login with session persistence
- Custom expense categories with color coding
- Add expenses with date, amount, category and optional photo
- Filter expenses by date range
- Category spending reports with progress bars
- Monthly budget goals

## How to Run
1. Clone this repository
2. Open the project in Android Studio
3. Sync Gradle and run on an emulator or physical device
